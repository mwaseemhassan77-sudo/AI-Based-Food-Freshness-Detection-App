# 🍎 SafeBite - AI Food Freshness Detection App

SafeBite is an AI-powered mobile application that detects whether fruits and vegetables are **fresh or rotten** using **Artificial Intelligence, Deep Learning, and Computer Vision**.

The application allows users to scan food items using their smartphone camera and instantly receive freshness predictions, helping reduce food waste and improve food safety.

---

## 🚀 Features

- 🤖 AI-Powered Food Freshness Detection
- 📷 Real-Time Camera Scanner
- 🍎 Fruit Recognition
- 🥕 Vegetable Recognition
- ⚡ Fast Prediction Results
- 📱 Mobile-Friendly Interface
- 🧠 Deep Learning Based Classification
- 🌐 Offline AI Model Support
- 📊 Fresh vs Rotten Detection
- 🔍 Computer Vision Technology

---

## 📱 How It Works

1. Open the SafeBite Mobile App
2. Click the Scanner Icon
3. Scan a Fruit or Vegetable
4. AI Analyzes the Food Item
5. Get Instant Freshness Results
